Chat sencillo usando SillyClient. El usuario, en las pestañas de arriba,
puede seleccionar su nombre y room a la que quiere conectarse. Debajo tiene
el input y botón para escribir y enviar mensajes a la sala. Cuando un usuario entra
a una sala en la que ya hay mensajes enviados, el usuario más antiguo envia todos
los mensajes de los que dispone al usuario que acaba de entrar para que este pueda
leer los mensajes anteriores.

Se pueden enviar mensajes tanto pulsando el botón con el ratón como presionando
la tecla enter.
